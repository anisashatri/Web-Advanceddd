HTML

HTML is the standard markup language for creating Web pages. HTML represents the structure and the content of a web page. Acronym of HTML is Hypertext Markup Language. Hypertext refers to text which might contain links (hyperlinks) to other texts in a HTML page. Markup language refers to the way tags are used to define the structure and content within the page.
